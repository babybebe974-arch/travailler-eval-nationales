🔗 1. Accéder à votre Google Sheet

✎ Rappel : Tous les résultats des élèves (diagnostic, entraînements français/maths) sont automatiquement envoyés vers le Google Sheet que vous avez configuré avec le script Apps Script.
📌 Pour consulter les données en direct :
  1. Ouvrez le Google Sheet que vous avez utilisé lors du déploiement du script.
  2. L’onglet Resultats contient toutes les lignes avec : date, prénom, matière, compétence, exercice, réponse, succès, etc.
  3. Vous pouvez appliquer des filtres (par prénom, par compétence, par matière) pour analyser les progrès.
🔁 Si vous avez perdu le lien : Rendez-vous dans votre Google Drive, recherchez le fichier intitulé “Résultats évaluations nationales” (ou le nom que vous lui avez donné).

📎 2. Intégrer un aperçu du Sheet dans cette page (optionnel)

💡 Astuce : Vous pouvez afficher directement votre Google Sheet ci-dessous en le partageant en lecture seule.
➊ Partagez votre Sheet :
  • Dans votre Google Sheet, cliquez sur « Partager » (en haut à droite).
  • Modifiez l’accès : « Toute personne disposant du lien » → rôle « Lecteur ».
  • Copiez le lien (il ressemble à https://docs.google.com/spreadsheets/d/XXXXXXXXXXXXXX/edit?usp=sharing).

📈 3. Exploiter les données – conseils

📋 Colonnes disponibles : DateHeure, Prenom, Matiere, Competence, ExerciceId, ReponseDonnee, Correct, ScoreMax, TempsMs
📊 Exemples d’analyses rapides (dans Google Sheets) :
  • Filtrer par Matiere = francais pour voir les résultats en français.
  • Filtrer par Correct = NON pour identifier les compétences non maîtrisées.
  • Créer un tableau croisé dynamique : Lignes = Prenom, Colonnes = Competence, Valeurs = Correct (moyenne).
  • Graphique d’évolution : utiliser la colonne DateHeure pour suivre les progrès dans le temps.
📥 Télécharger les résultats : Dans votre Sheet, allez dans Fichier → Télécharger → CSV pour importer dans un autre logiciel.

⚠️ 4. Confidentialité

Les prénoms sont stockés dans le Sheet. Si vous souhaitez anonymiser davantage, remplacez les prénoms par des codes dans la colonne Prenom du Sheet (les futures entrées utiliseront les prénoms saisis par les élèves). Pensez à supprimer les anciennes lignes sensibles.
